Локальный Base64 и Base64url Кодировщик/Декодировщик

Python программа для кодирования и декодирования строк в формате Base64 и Base64url. Этот инструмент предоставляет простой способ преобразовать текст в формат Base64 или Base64url и наоборот:
1. Поддержка нескольких языков.
2. Поддержка различных кодировок символов (UTF-8, ASCII, ISO-8859-1, UTF-16).
3. Кроссплатформенная совместимость.

![conver_base64-v.2](https://github.com/user-attachments/assets/ea174ecc-c330-46da-b4b4-747f99d4178f)


Сборка программы в исполняемый файл для запуска в операционной системе Microsoft Windows без необходимости устанавливать Python (но для компиляции, всё же python потребуется но один раз), для этого в командной строке windows перейдем в директорию с скаченным файлом [conver_base64-v.2.py и f1-ks.ico] и выполни команду:

  >python -m PyInstaller --onefile --windowed --icon=f1-ks.ico conver_base64-v.2.py

далее рядом с файлом [conver_base64-v.2.py] появится ещё один файл [conver_base64-v.2.spec] откроем этот файл любым текстовым редактором, поправим в нём одну строку:

a = Analysis(
...
    datas=[('f1-ks.ico','.')], # Вот тут добавляем ('f1-ks.ico','.')
... )

и сохраним изменения.
далее запускаем ещё раз сборку программы, но уже с файла [conver_base64-v.2.spec]:

  >python -m PyInstaller conver_base64-v.2.spec

после сборки рядом с файлами py и spec появится папка dist, в этой папке будет исполняемы файл готовый к работе.

PS
Возможно, потребуется установка пакетов (если они ранее у вас не устанавливались):
